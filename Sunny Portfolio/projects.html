BMI CALCULATOR

This project is a Body Mass Index (BMI) Calculator web application. It allows users to input their height and weight to calculate their BMI, which is a widely used measure to determine whether an individual is underweight, normal weight, overweight, or obese. The application also provides a visual indication based on the BMI value, displaying an appropriate image corresponding to the BMI category.
